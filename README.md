# qgama-0.2.4

Qgama version 0.24 was originally written by Jiří Novák as a part of his master thesis at the Department of Mapping and Cartography 
at the Czech Technical University in Prague.
* https://dspace.cvut.cz/handle/10467/8322

Following development of the project was independent of his code and is hosted at
* https://www.gnu.org/software/gama/
* https://ftp.gnu.org/gnu/gama/gama-qt/
* http://git.savannah.gnu.org/cgit/gama/qt.git

Binary builds for Windows are available from
* https://sourceforge.net/projects/qgama
* https://sourceforge.net/projects/gnu-gama-local/
